# Harapan Istana Cleaning Service (HARIS) Webpage

This is the official landing page project for Harapan Istana Cleaning Service (HARIS), a Surabaya-based cleaning service brand. The project is built using:
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v3)

### 1. Install Dependencies
Open your terminal in the project directory and run:

```bash

```

### 2. Start the Development Server
This will start a local server at `http://localhost:5173` (or similar).

```bash
npm run dev
```

### 3. Build for Production
To create an optimized build for deployment (e.g., Vercel, Netlify):

```bash
npm run build
```

The output will be in the `dist/` folder.
